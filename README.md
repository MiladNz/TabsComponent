###  **Customization**
You can easily update the **tabs data** inside `App.jsx`:
```jsx
const tabs = [
  { title: "TAB 1", content: "This is the content for Tab 1" },
  { title: "TAB 2", content: "This is the content for Tab 2" },
  { title: "TAB 3", content: "This is the content for Tab 3" },
];
```
